# vue-web-lab

Reference Vue 3 + Vite sandbox for BeeNode WASM.

## Prereqs

- `pkg/` must exist at the repo root (generated by `wasm-pack build --target web`).
- Recommended: `../scripts/build-wasm.sh` (ensures `--no-default-features --features wasm`).

## Run

```bash
cd vue-web-lab
npm install
npm run dev
```

## Notes

- The app imports `@beenode-wasm/beenode.js` from `../pkg`.
- Vite is configured to allow access to `../pkg` during dev.

## Shell Contract

See `contracts/shell-contract.json` and `docs/SHELL_CONTRACT.md` for cross-platform parity fixtures.

## Magic link (dev vs prod)

Environment switches:

```
VITE_MAGIC_LINK_MODE=dev
VITE_MAGIC_LINK_BASE_URL=https://auth.example.com
```

- `dev`: local stub prints a magic link to console.
- `prod`: calls `/auth/magic/request` + `/auth/magic/verify` on the base URL.
